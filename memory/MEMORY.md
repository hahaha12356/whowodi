## 🎯 关键问题记录

### 2026-02-08 并发领取词语Bug

**问题**：两个用户同时点击"领取词语"时，会领到相同号码

**原因**：客户端无法保证原子性操作

**解决方案**：使用云函数 + 数据库事务

**详情**：[memory/BUGS.md](memory/BUGS.md)

---

## 💡 开发建议

1. 遇到并发问题，优先考虑云函数
2. 客户端的"检查-创建"永远不是原子的
3. 使用数据库事务确保数据一致性
4. 详细的部署文档对团队协作很重要
