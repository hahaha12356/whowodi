# 词语分配助手 - 快速开始指南

> 一款基于uni-app开发的微信小程序，用于聚会活动中的词语随机分配工具。

## ✨ 项目特性

- 🚀 **快速配置**: 选择人数，自动推荐最优配置
- 🎯 **自定义配置**: 灵活调整A/B/C类人数、选择词库
- 🎲 **公平算法**: Fisher-Yates洗牌算法确保随机性
- 📱 **纯前端**: 无需服务器，数据本地存储
- 🔒 **隐私保护**: 24小时自动清理，不上传用户数据
- 📊 **实时统计**: 查看进度实时更新

## 🛠️ 技术栈

- **框架**: uni-app (Vue 3)
- **平台**: 微信小程序
- **样式**: SCSS
- **存储**: uni.storage API

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 开发模式

```bash
npm run dev:mp-weixin
```

### 3. 微信开发者工具

1. 打开微信开发者工具
2. 导入项目：选择 `dist/build/mp-weixin` 目录
3. 配置AppID（测试可使用测试号）
4. 点击"编译"即可预览

### 4. 生产构建

```bash
npm run build:mp-weixin
```

## ❓ 常见问题

### Q: 方案链接失效怎么办？
A: 方案有效期为24小时，超过时间会自动失效。请重新生成新方案。

### Q: 如何确保分配的公平性？
A: 使用Fisher-Yates洗牌算法，保证每个位置被分配的概率相等。

### Q: 支持多少人参与？
A: 支持3-20人，建议6-12人体验最佳。

### Q: 数据会被保存吗？
A: 所有数据仅保存在本地，24小时后自动清除，不会上传服务器。
